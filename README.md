# Portfólio estático (React + Vite)

Projeto de portfólio pessoal com foco em quatro áreas:

- Desenvolvimento
- Ilustração
- Música
- Desenvolvimento de jogos

## Executar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Deploy na Vercel

1. Suba este repositório para o GitHub.
2. Na Vercel, clique em **Add New > Project**.
3. Importe o repositório.
4. Mantenha os padrões do Vite:
	- Build Command: `npm run build`
	- Output Directory: `dist`
5. Clique em **Deploy**.

## Personalização rápida

- Dados pessoais e links: `src/profile.js`
- Cores e layout: `src/App.css` e `src/index.css`
